# 雪茄品鉴视频创作 Skill 设计

## 目标

创建个人 Codex Skill `creating-cigar-review-videos`，用于生成 2–3 分钟的暗金奢华风格雪茄品鉴短视频完整制作方案。技能同时覆盖策划、脚本、拍摄、剪辑和发布，并采用用户资料优先、联网资料补充的混合模式。

## 使用场景

当用户提供雪茄名称、图片、规格、个人品鉴记录或参考视频，并要求创作雪茄测评、品鉴、介绍或暗金风格短视频时触发。

## 技能结构

技能默认安装到个人技能目录 `C:\Users\Administrator\.codex\skills\creating-cigar-review-videos`。

- `SKILL.md`：定义输入检查、资料补充、内容生成和质量检查流程。
- `references/cigar-review-framework.md`：记录雪茄品鉴维度、术语和评分逻辑。
- `references/dark-gold-style.md`：记录暗黑奢华视觉、灯光、构图、字幕和调色规范。
- `references/output-template.md`：定义脚本、分镜、字幕、评分表、封面和发布文案格式。
- `references/research-rules.md`：定义来源优先级、事实核实和主观评价标注规则。
- `agents/openai.yaml`：提供技能列表所需的展示名称、简介和默认提示。

首版不加入程序脚本或固定媒体素材，避免输出过度模板化，并保留针对不同雪茄和创作者条件进行调整的空间。

## 输入

优先接收以下信息：

- 雪茄品牌、系列和具体型号。
- 尺寸、产地、茄衣、茄套和茄芯资料。
- 产品照片、包装照片或参考视频。
- 用户自己的冷闻、前段、中段、后段和余味记录。
- 目标受众、账号定位、拍摄器材与场地限制。

允许输入不完整。缺失资料应进入核实流程，而不是由模型补写为事实。

## 执行流程

1. 解析用户目标和现有资料。
2. 检查品牌、系列、尺寸、产地和结构等关键字段。
3. 对缺失的客观资料执行联网查询，优先采用品牌方、生产商或可靠专业零售商资料。
4. 将内容划分为客观资料、用户主观品鉴和创作建议三类。
5. 设计 2–3 分钟视频的开场、章节、评分和结论结构。
6. 生成完整创作包，并根据用户器材和场地调整执行难度。
7. 检查事实归属、时长、结构、视觉一致性和输出完整性。

## 输出契约

每次默认输出以下部分：

1. 创作定位：目标受众、核心卖点和风格关键词。
2. 标题方案：作品标题、封面标题和开场钩子。
3. 时间轴脚本：按时间区段组织的完整 2–3 分钟结构。
4. 口播稿：适合自然表达的完整旁白。
5. 字幕稿：比口播更精炼，突出关键词和结论。
6. 分镜表：景别、画面、机位、灯光、道具和建议时长。
7. 品鉴评分：香气、浓郁度、复杂度、平衡度、燃烧表现和余味。
8. 视觉包装：暗金配色、品牌区、章节卡、评分表和字幕规范。
9. 后期方案：剪辑节奏、转场、调色、音乐和音效方向。
10. 发布素材：简介、话题标签和互动引导。
11. 事实说明：已核实资料、用户主观评价和待确认信息。

## 视觉方向

- 使用黑、深棕和琥珀金作为核心色彩。
- 使用单侧暖光、低照度环境和微距特写表现雪茄与包装质感。
- 在画面顶部或固定区域保留品牌标识位置。
- 使用细金线章节卡和评分表建立专业品鉴感。
- 使用白色正文字幕，金色或红色强调关键词和分数。
- 保持中慢节奏，让产品细节和品鉴信息有足够停留时间。

## 内容与事实边界

- 客观规格必须能够追溯至可靠来源。
- 用户提供的品鉴感受应标记为个人体验。
- 模型建议的口感描述应标记为创作建议，不得伪装为实际品鉴结果。
- 来源冲突时列出差异，不擅自选取未经验证的结论。
- 资料不足时输出待补充项或使用明确的占位表达。
- 内容聚焦合法成年人雪茄文化和品鉴，不添加健康功效或安全性暗示。

## 错误处理

- 无法确认具体型号时，先要求补充包装、茄标或完整名称。
- 无法联网或来源不足时，仅使用用户资料，并明确限制。
- 用户没有实际品鉴记录时，不生成伪装成亲测体验的确定性评价。
- 用户器材不足时，将复杂灯光和运镜降级为可执行的手机拍摄方案。

## 验证标准

- 使用至少一个完整雪茄案例检查输出契约的所有部分。
- 验证时间轴总时长位于 2–3 分钟范围。
- 验证客观事实、个人体验和创作建议清晰分离。
- 验证暗金视觉风格贯穿灯光、构图、字幕、图表和调色建议。
- 验证技能在资料完整和资料缺失两种输入下均能给出可用结果。
- 使用 `quick_validate.py` 检查技能目录、YAML frontmatter 和命名规范。

## 非目标

- 不生成或宣称未经用户提供的真实吸食体验。
- 不替代专业健康建议，也不宣传雪茄具有健康益处。
- 不在首版自动生成视频文件、图片素材或剪辑工程。
- 不绑定单一品牌、单一账号名称或固定 Logo。
